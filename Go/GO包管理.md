### go mod 
- goland使用go mod 
1. 首先下载安装vgo 
```
官方仓库：https://github.com/golang/vgo
克隆到　src/goland.org/x/
安装　go install gloand.org/x/vgo
```
2. goland 安装　vgo 支持插件

3. goland 配置　go module(vgo)  选项
vgo  location  选项会自动识别　vgo　sdk路径
